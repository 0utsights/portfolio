## This is my personal portfolio website, built from scratch using plain **HTML, CSS, and JavaScript**.

The site is fully static and hosted on **GitHub Pages**.

## Live Site

https://johnsurles.com

## Tech Stack

- **HTML** – page structure
- **CSS** – layout, typography, and layering
- **JavaScript (Canvas API)** – animated hex grid background
- **GitHub Pages** – hosting
- **Namecheap** – custom domain + DNS

---

Concepts used to mathematically draw frames and create the procedurally generated infinite hex grid can be found on https://www.redblobgames.com/grids/hexagons/

## Key concepts used:
- Axial hex coordinates
- Pointy-top hex geometry
- Camera-based movement instead of moving tiles
- Seamless looping via modular wrapping
